Running BroadViewAgent and BroadViewBstApp

The BroadViewAgent and the BroadViewBstApp communicate over HTTP. Each of the application maintains respective configuration files. The default location of configuration file has to be in the same directory where the application is present. Follow the steps mentioned below, to start BroadViewAgent.

1.cd to directory where BroadViewAgent is located

2.Copy the agent_config.cfg created earlier

# scp <username>\<server_ip>:<file_path>/agent_config.cfg

3.Start BroadViewAgent

# ./BroadViewAgent &

Follow the steps, mentioned below, to start BroadViewBstApp.

1.cd to directory where BroadViewBstApp is located

2.Copy the bstapp_config.cfg created earlier

3.Start BroadViewBstApp
